# Async Listing with Svelte/Webpack

## Deploy steps

1. Pull the code and open project directory
2. Execute composer install
```bash
composer install
```
3. Build the project with 
```bash
npm run build
```

4. Use project entry point as 
```bash
public/index.html
```
## Automated Testing

### PHPUnit tests:
1. Unit
```bash
npm run test-unit
```

2. Functional
```bash
npm run test-functional
```
